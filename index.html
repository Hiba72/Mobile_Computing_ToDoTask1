How to use Github

1. Install git on your system.

Go to https://git-scm.com/downloads. Download suitable git version for your OS.

2. Make an account on Github.com.

If you already have an account then simply sign in otherwise sign up.

2. Create a new repository.

4. Make html page on VS code.

Goto File -> Open folder. Then select the folder and create html page in it. It will look like the below image.

5. Perform these commands on command line.

Just for the refrence I am attaching screenshot of my command line.

Now when we have to change/update anything in our code we only need to perform these commands:

1. git add .

2. git commit -m "--message--" .

3. git push

4. Upload a web page.

Go to the settings tab.

Now go to pages.

Now select main branch.

Save and then you can see site is published. When you click on it your web page will be displayed.
